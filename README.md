## 如果你想使用 Rhythm ...

目前，Rhythm 是驱动 [摸鱼派](https://fishpi.cn) 的社区系统，一些源代码、文案及元素可能直接涉及摸鱼派的内容，我们目前还未做面向用户的适配（即用户版 Rhythm）,如果是小白用户，还请耐心等待。如果你有一定的编程基础，可以尝试自定义 Rhythm。
